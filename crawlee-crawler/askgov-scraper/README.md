# AskGov Scraper

This is a Playwright-based web scraper built with [Crawlee](https://crawlee.dev/) designed to extract Q&A data from AskGov (specifically the ECDA topic as a starting point). It runs locally, uses an Apify proxy (to bypass blocks like Cloudflare), and outputs the results into a single JSON file.

## Prerequisites

- [Node.js](https://nodejs.org/en/) (v16 or higher)
- [npm](https://www.npmjs.com/) (comes with Node.js)
- An [Apify](https://console.apify.com/) account (to get a Proxy password if you need to bypass WAFs/Cloudflare).

## Getting Started

1. **Install Dependencies:**
   Run the following command in the project directory to install all necessary packages, including the Playwright browsers:
   ```bash
   npm install
   ```

2. **Environment Variables:**
   A `.env` file is required to store your Apify Proxy credentials. 
   - Copy the provided `.env-example` to `.env`:
     ```bash
     cp .env-example .env
     ```
   - Open `.env` and replace the placeholder with your actual **Apify Proxy Password** (Note: this is usually different from your Apify API Token. You can find it in the Apify Console under the Proxy tab).

3. **Running the Scraper:**
   Start the crawler with the following command:
   ```bash
   npm start
   ```
   

## Output

Once the scraper has finished processing the pagination and extracting the questions and answers, the extracted data is automatically saved into a single file at:
`storage/output.json`

## Running Without a Proxy

If AskGov is not currently blocking your IP and you wish to run the scraper without using the Apify Proxies:
1. Open `src/main.js`.
2. Locate the line where `proxyConfiguration` is passed into the `PlaywrightCrawler` instance options (around Line 11).
3. Comment out `proxyConfiguration,` like so:
   ```javascript
   const crawler = new PlaywrightCrawler({
       // proxyConfiguration, // comment out to not use proxy
       // ...
   });
   ```
4. Run `npm start`.

## Project Structure

- `src/main.js`: Contains the main crawling and extraction logic. The `requestHandler` captures the Q&A text and manages pagination ("View more" buttons).
- `storage/`: Auto-generated by Crawlee. It holds the request queues, Key-Value stores, and the final `output.json`. This folder is ignored by git.
- `.env`: (Ignored by git) Houses your private configuration credentials.
- `.env-example`: A template showing the required environment variables for the project.
